### 🌐 Week 2: Web Architectures & API Contracts
*Focus: Building robust UI state machines and designing clean server-client serialization schemas.*

### 📊 Week 3: Directed Graphs & Pipeline Optimization
*Focus: Processing structural networks, detecting loops, and designing execution dependencies.*
* **Block & Graph Systems:** Analyzed block-based pipeline flow architectures mapping inputs to designated processing outputs.
* **Acyclic Validation (DAGs):** Learned the mathematics behind Directed Acyclic Graphs (DAGs) to identify critical loop vulnerabilities or circular logic in execution sequences.
* **Topological Sorting:** Applied topological sorting algorithms (`NetworkX`) to successfully resolve execution hierarchies, ensuring dual-dependent functions run safely in sequential order.

---

### 📷 Week 4: Low-Level Image Processing & Matrix Mechanics
*Focus: Handling visual data as structured numerical arrays and understanding pixel transformations.*
* **Images as Coordinate Arrays:** Conceptualized multi-channel image layouts as raw matrices (height × width × channels) using OpenCV standards.
* **Array Transformations:** Covered the mathematical concepts behind low-level spatial resampling, axis resizing, and matrix slicing (region cropping).
* **Color Space Remapping:** Analyzed channel extraction and pixel intensity transformations required to downsample color arrays into single-channel grayscale maps without compromising structural clarity.
